About Us

Ghost Chat is an anonymous, region-based realtime chat application with room codes for private rooms, emoji support, presence, and a clean, responsive UI with light/dark modes.

Technologies Used

  • Flask, Flask‑Login, Flask‑SocketIO
  • SQLAlchemy (SQLite/PostgreSQL)
  • Bootstrap 5, Bootstrap Icons
  • WTForms, email‑validator
  • Eventlet (production Socket.IO)

What it does

  • Anonymous regional chat rooms (General, Sports, Tech, Love, Dating)
  • Create private rooms with unique codes; join via code
  • Real‑time messages, typing indicators, emoji insert
  • Presence: shows online users per room in real time
  • Profiles: name, gender, age, height, interests
  • Dark mode by default, responsive UI
